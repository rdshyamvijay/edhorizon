# Design System (Playful & Premium)

## Core Aesthetic
- **Sidebar**: Deep charcoal/black (`#111111`) for maximum contrast.
- **Background**: Soft off-white/beige (`#F8F7F4`) in light mode; Deep obsidian in dark mode.
- **Cards**: Pure white with soft, large shadows (`shadow-xl`) and high border radius (`rounded-3xl`).
- **Accents**: High-energy gradients (Indigo-to-Violet, Pink-to-Orange).

## Color Palette (Light Mode)
- **Background**: `#F8F7F4` (Soft Beige)
- **Sidebar**: `#111111` (Matte Black)
- **Primary Gradient**: `from-indigo-600 to-violet-600`
- **Secondary Gradient**: `from-orange-500 to-rose-500`
- **Cards/Popovers**: `#FFFFFF`
- **Text**: `#1A1A1A` (Primary), `#6B7280` (Muted)

## Color Palette (Dark Mode)
- **Background**: `#0A0A0A` (Obsidian)
- **Sidebar**: `#000000` (Pitch Black)
- **Cards**: `#161616` (Elevated Black)
- **Text**: `#F3F4F6` (Primary), `#9CA3AF` (Muted)

## Typography
- **Primary Font**: Plus Jakarta Sans (Geometric, friendly, modern).
- **Headings**: Plus Jakarta Sans (Extra Bold), tracking-tighter.
- **Body**: Plus Jakarta Sans (Medium) for refined clarity.

## Layout & Components
- **Sidebar**: Slim, minimal icons. Active state is a white/indigo pill.
- **Progress Bars**: Thick, vibrant gradients with soft gray track.
- **Buttons**: Extra rounded (`rounded-full`), subtle hover lift, gradient options for primary actions.
